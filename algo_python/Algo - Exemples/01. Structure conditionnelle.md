# Structure conditionnelle `Si... Sinon...`

```
Si (condition) Alors:
    # Instructions
Sinon:
    # Instructions
```

# Algorithme de température

```algo
Algo température

temp : nombre

Début
    Lire(temp)

    Si temp <= 0 Alors:
        Ecrire("C'est de la glace" )

    Si temp > 0 ET temp < 100 Alors:
        Ecrire("C'est du liquide")
        
    Si temp >= 100 Alors:
        Ecrire("C'est de la vapeur" )
Fin
```
